     * <p>

     * <p>
     * &lt;div&gt;

     * <p>
     * &lt;form&gt;
     * &lt;header&gt;
     * &lt;footer&gt;
     * &lt;section&gt;
     * <p>
     * Inline Elements
     * <p>
     * An inline element does not start on a new line and only takes up as much width as necessary.
     * <p>
     * This is an inline &lt;span&gt; element inside a paragraph.
     * <p>
     * Examples of inline elements:
